# Free stock-quote forum signature

This repository generates one direct PNG image containing only:

- ticker
- current/delayed stock price
- daily percentage change

Example:

`NVDA   $171.38   ▲ +2.14%`

The image is regenerated by GitHub Actions approximately every five minutes.

## 1. Create a free GitHub account

Go to GitHub and create an account if needed.

## 2. Create a PUBLIC repository

Recommended repository name:

`stock-signature`

The repository must be public for unlimited free use of standard GitHub-hosted Actions runners and for a public raw image URL.

## 3. Upload every file in this package

Preserve this folder structure:

```text
.github/
  workflows/
    update-signature.yml
generate_signature.py
requirements.txt
tickers.txt
stock_signature.png
README.md
```

## 4. Choose your tickers

Edit `tickers.txt`. Put one ticker on each line:

```text
NVDA
TSLA
PLTR
```

For one ticker, leave only one line.

Common examples:

```text
SPY
QQQ
NVDA
TSLA
PLTR
```

Yahoo-style ticker symbols are used. Examples:

- Berkshire Hathaway Class B: `BRK-B`
- S&P 500 index: `^GSPC`
- Bitcoin in U.S. dollars: `BTC-USD`

## 5. Enable and test GitHub Actions

Open the repository's **Actions** tab.

Select **Update stock signature**, then choose:

**Run workflow → Run workflow**

After it finishes, confirm that `stock_signature.png` has been updated.

## 6. Use the direct PNG URL

Your direct image URL will be:

```text
https://raw.githubusercontent.com/YOUR-USERNAME/stock-signature/main/stock_signature.png
```

Replace `YOUR-USERNAME` with your GitHub username.

If you chose a different repository name, replace `stock-signature` too.

## 7. Put it in the KAWF signature

HTML form:

```html
<img src="https://raw.githubusercontent.com/YOUR-USERNAME/stock-signature/main/stock_signature.png" alt="Stock quotes">
```

BBCode form, if KAWF uses BBCode:

```text
[img]https://raw.githubusercontent.com/YOUR-USERNAME/stock-signature/main/stock_signature.png[/img]
```

## Refresh and caching notes

GitHub runs scheduled workflows approximately every five minutes, but scheduled jobs can be delayed.

The forum or browser may cache the image. The raw URL stays the same while the PNG contents change. A hard refresh may be needed when testing.

GitHub can automatically disable scheduled workflows in a public repository after 60 days without repository activity. Re-enable the workflow from the Actions tab if that occurs.

## Quote-data note

The script uses Yahoo Finance's unofficial chart endpoint without a login or API key. It is suitable for a casual forum signature, not for trading decisions. Availability, delays, fields, and access can change without notice.

## Appearance changes

The top of `generate_signature.py` contains editable settings:

- `WIDTH`
- `ROW_HEIGHT`
- `FONT_SIZE`
- background and text colors

The default format is dark, compact, and approximately 48 pixels high per ticker.
